# Firebase Hosting

`node - v12.4.0`

## Get Started

### Install

1. Set environment variables.
    ```python
    # .env
    FIREBASE_API_KEY=""
    FIREBASE_DATABASE_URL=""
    FIREBASE_PROJECT_ID=""
    ```


### Start & Build

1. Start with Webpack Development Server.
    ```bash
    $ npm run start
    ```

2. Build with Webpack.
    ```bash
    $ npm run build
    ```
    - this will build the application in `dist/`.


### Deploy

2. Deploy to .
    ```bash
    $ npm run deploy
    ```
